# 1downy

## Repositories
- [ADsBlocking](https://github.com/1downy/ADsBlocking) — HTML/CSS frontend (ad-blocking pages).
- [PateronScraper](https://github.com/1downy/PateronScraper) — Python scraper.
- [The-Sims-Resource-Downloader-enhanced](https://github.com/1downy/The-Sims-Resource-Downloader-enhanced) — Python downloader for The Sims Resource.
- [SimDatabase](https://github.com/1downy/SimDatabase) — Python data project.
- [GraphiXpertise](https://github.com/1downy/GraphiXpertise) — Python + HTML/CSS web project.
- [GPU-dataset](https://github.com/1downy/GPU-dataset) — GPU-related dataset.
- [MS-Office-Activator-2019](https://github.com/1downy/MS-Office-Activator-2019) — Batch scripts.
- [1downy](https://github.com/1downy/1downy) — Profile config files.

## Languages
Mostly Python, HTML/CSS, and Batch scripts.
